# 🛡️ AI-Robot-Trap: The Infinite Maze
An AI-powered cybersecurity deception tool built on Kali Linux.

## 🚀 Features
- **Behavioral Analysis:** Uses Google Gemini AI to spot bot-like clicking patterns.
- **Active Deception:** Traps detected bots in an infinite "Vault Maze" to waste their time.
- **Live Logging:** Tracks "Intruder Levels" in a local database.

## 🛠️ Setup
1. Clone this repo.
2. Install requirements: `pip install flask google-genai requests`
3. Add your Gemini API Key in `ultimate_trap.py`.
4. Run: `python3 ultimate_trap.py`
